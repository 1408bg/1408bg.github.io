# C_Express

## express.js in C
해당 헤더 파일은 express의 일부 기능을 그것과 유사한 문법으로 사용할 수 있도록 합니다.

## how to use
당신의 프로젝트에 두 개의 헤더 파일을 추가하고 그중 `express.h`를 include 하세요.

## simple, light
그저 한번의 include로 모든 함수와 종속성을 관리하세요, 파일 하나에 그 모든 것이 들어있습니다.
