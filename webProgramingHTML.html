기본 입력

  • 텍스트 입력을 위한 태그를 정확히 사용할 수 있는가?

    1. 헤딩

      • h1 - h6의 범위를 가지며 숫자가 낮을 수록 크기가 커진다.

    2. p

      • 매우기본적인텍스트입력태그
      • display: block이다.

    3. span

      • display: inline이다.

    4. pre

      • 친거(띄어쓰기, 줄바꿈) 그대로 나온다.

    5. 강조 태그

      • strong, b
        굵어진다.
      • em, i
        기울어진다.

    6. blockquote

      • 인용, 자동으로 탭쳐준다.

  • 목록의 종류와 특징을 이해하고 사용할 수 있는가?

    1. ol

      • 순서 있는 리스트
      • type, start, reverced 속성이 있다.

    2. ul

      • 순서 없는 리스트

    3. dl

      • 제목 있는 리스트
      • dt로 제목, dd로 요소를 쓴다.

    4. li

      • 리스트 요소
      • dl을 제외한 모든 리스트 요소다.

  • 표를 만들기 위한 태그를 적절히 사용할 수 있는가?

    1. table

      • 이안에 테이블 집어넣는다.
        지금부터 나오는건 다 table안에 들어간다.
      • border 속성으로 테두리 넣을 수 있다.

    2. caption

      • 표 제목이다.

    3. tr

      • 표의 한 줄이다.
      • th로 제목 셀, td로 일반 셀을 집어넣는다.

    4. 셀 확장 속성

      • colspan: 열 합치기
      • rowspan: 행 합치기

  • 이미지, 오디오, 비디오 등을 정확히 삽입할 수 있는가?

    1. img

      • 빈태그
      • src 속성으로 경로 지정
      • alt 속성으로 대체 텍스트 지정
      • width 속성으로 가로 길이 조정 (px 쓰지 말자)

    2. audio

      • src 속성으로 경로 지정
      • 태그 안에 대체 텍스트 작성
      • controls 속성으로 제어 인터페이스 표시
      • loop, muted, autoplay, preload 속성을 가짐

    3. video

      • audio랑 동일

    4. object

      • data 속성으로 경로 지정
      • width, height 속성 보유

    5. embed

      • src 속성으로 경로 지정

  • 외부 사이트로 연결하는 링크를 삽입할 수 있는가?

    1. a

      • href 속성으로 경로 지정
      • target="blank"로 새 창 열기
      • a 안에 집어넣은 태그는 다 하이퍼링크 적용 대상

    2. 절대 경로와 상대 경로

      • 절대 경로: /, root가 기준 (http 포함)
      • 상대 경로: ./(생략가능), 현재 위치가 기준
        ../ 상위 폴더 이동, / 하위 폴더 이동

폼 삽입

  • 문제 상황에서 요구하는 폼 요소를 삽입할 수 있는가?

    1. form

      • 이 안에 폼 집어넣는다.
        앞으로 나오는 태그는 다 이 안에 들어간다.
      • method로 전송 타입, action으로 라우팅 주소를 지정한다.

    2. fieldset

      • form 내부 요소를 분리하는 태그
      • 이 안에 input 집어넣는다.

    3. legend

      • fieldset의 제목을 나타낸다.

    4. label

      • input의 제목을 나타낸다.
      • 이 안에 input 집어넣어도 되고
        for 속성으로 input이랑 id 맞춰도 된다.

    5. input

      • 여기에서 사용자 입력 받는다.
      • display: inlineblock이다.
      • 텍스트 입력
        type="text" :
      • 확인, 초기화 (value로 안에 내용 변경)
        submit, reset :
      • 라디오, 체크박스 (name 꼭 맞추기)
        radio, checkbox :
      • autofocus, placeholder, readonly, required 속성 보유

    6. textarea

      • 여러줄 입력받기
      • 열 rows, 행 cols 속성 보유

    7. select

      • 저걸로 드롭다운 만들고
      • option 태그로 선택지 지정
